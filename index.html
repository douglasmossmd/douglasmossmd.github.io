ED Physician Productivity Dashboard

An interactive web dashboard built for the UChicago Medicine Center for Care and Discovery (CCD) Emergency Medicine department to surface and communicate physician productivity data to department leadership and individual providers. The system integrates real-time scheduling data from ShiftAdmin with billing data from the UCM Profee Cube to calculate three core metrics — wRVUs per Hour, Encounters per Hour, and wRVUs per Encounter — placing each provider into a four-quadrant framework based on throughput and encounter complexity. Built with Python and Streamlit, it features provider-level drill-down, time period filtering, shift activity breakdowns, benchmarking against internal group percentiles, hover-over metric explanations with data source citations, and an automated personalized PDF report pipeline for delivery to individual providers.
